# Third-Party Notices and Asset Licensing

This repository includes audio files used by the game prototype.

## Important

The MIT License in `LICENSE` applies to the game code and project documentation created for **Adventures from Udder Space**.

It does **not automatically relicense third-party music or sound effects**. Before making this repository public or embedding the game on the TNT website, verify that every included audio file is cleared for public web use.

## Recommended Release Rule

If you are unsure about a music track's license, do not publish it publicly yet. Replace it with:

- original music,
- clearly licensed royalty-free music,
- public-domain music with verified source documentation, or
- procedural/in-game generated audio.

Keep a copy of each asset's license or source URL in this file before public release.
